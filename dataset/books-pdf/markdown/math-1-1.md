### **第一章：平面直角坐标系中的直线**

本章的核心是使用代数方法（坐标和方程）来研究直线的几何性质和关系。

#### **1.1 直线的倾斜角与斜率**
*   **倾斜角 (Inclination Angle, θ)**：指直线与x轴正方向所成的最小正角，取值范围是 `[0, π)`。
*   **斜率 (Slope, k)**：当倾斜角 `θ ≠ π/2` 时，斜率 `k = tan(θ)`。当直线与x轴垂直时，倾斜角为 `π/2`，斜率不存在。
*   **斜率公式**：经过两点 `P₁(x₁, y₁)` 和 `P₂(x₂, y₂)` 的直线斜率为 `k = (y₂ - y₁) / (x₂ - x₁) ` (其中 `x₁ ≠ x₂`)。

#### **1.2 直线的方程**
本节介绍了描述直线的不同形式的代数方程。
*   **点斜式 (Point-Slope Form)**：`y - y₀ = k(x - x₀)`，表示经过点 `(x₀, y₀)` 且斜率为 `k` 的直线。
*   **斜截式 (Slope-Intercept Form)**：`y = kx + b`，其中 `k` 是斜率，`b` 是直线在y轴上的截距。
*   **两点式 (Two-Point Form)**：`(y - y₁) / (y₂ - y₁) = (x - x₁) / (x₂ - x₁)`，表示经过两点 `(x₁, y₁)` 和 `(x₂, y₂)` 的直线。
*   **一般式 (General Form)**：`Ax + By + C = 0` (其中A, B不同时为零)。这是所有直线方程的通用形式。
*   **法向量 (Normal Vector)**：在一般式 `Ax + By + C = 0` 中，向量 `n = (A, B)` 是该直线的一个法向量，即与直线垂直。
*   **点法式 (Point-Normal Form)**：`A(x - x₀) + B(y - y₀) = 0`，表示经过点 `(x₀, y₀)` 且法向量为 `(A, B)` 的直线。

#### **1.3 两条直线的位置关系**
*   **位置关系判定**：
    *   **相交 (Intersect)**：斜率不等 `k₁ ≠ k₂`，或方程组有唯一解。
    *   **平行 (Parallel)**：斜率相等且y轴截距不等 `k₁ = k₂` 且 `b₁ ≠ b₂`，或方程组无解。
    *   **重合 (Coincident)**：斜率和y轴截距均相等 `k₁ = k₂` 且 `b₁ = b₂`，或方程组有无数解。
*   **垂直判定 (Perpendicularity)**：
    *   若斜率都存在，则 `k₁ * k₂ = -1`。
    *   对于一般式 `A₁x + B₁y + C₁ = 0` 和 `A₂x + B₂y + C₂ = 0`，垂直的充要条件是 `A₁A₂ + B₁B₂ = 0`。
*   **夹角公式 (Angle Formula)**：两条直线 `l₁` 和 `l₂` 的夹角 `α` 可通过其法向量 `n₁ = (A₁, B₁)` 和 `n₂ = (A₂, B₂)` 的夹角 `θ` 得到，公式为：
    `cos(α) = |cos(θ)| = |n₁ · n₂| / (|n₁| * |n₂|) = |A₁A₂ + B₁B₂| / (√(A₁² + B₁²) * √(A₂² + B₂²))`。

#### **1.4 点到直线的距离**
*   **点到直线距离公式**：点 `P(x₀, y₀)` 到直线 `Ax + By + C = 0` 的距离 `d` 为：
    `d = |Ax₀ + By₀ + C| / √(A² + B²)`。
*   **平行线间距离公式**：两条平行线 `Ax + By + C₁ = 0` 和 `Ax + By + C₂ = 0` 之间的距离 `d` 为：
    `d = |C₁ - C₂| / √(A² + B²)`。

### **第二章：圆锥曲线**

本章研究圆、椭圆、双曲线和抛物线，这些曲线都可以通过平面截取圆锥得到。

#### **2.1 圆**
*   **定义**：平面内与一个定点（圆心）的距离等于一个定长（半径）的点的轨迹。
*   **标准方程 (Standard Equation)**：`(x - a)² + (y - b)² = r²`，圆心为 `(a, b)`，半径为 `r`。
*   **一般方程 (General Equation)**：`x² + y² + Dx + Ey + F = 0`，其中必须满足 `D² + E² - 4F > 0`。
*   **直线与圆的位置关系**：通过联立方程后得到的二次方程的判别式 `Δ` 判断，或通过比较圆心到直线的距离 `d` 与半径 `r` 的大小判断。
*   **圆与圆的位置关系**：通过比较两圆心之间的距离 `d` 与两圆半径 `r₁`、`r₂` 的和或差的大小来判断（相离、外切、相交、内切、内含）。

#### **2.2 椭圆**
*   **定义**：平面内与两个定点 `F₁`, `F₂` (焦点) 的距离之和为常数 `2a` (且 `2a > |F₁F₂|`) 的点的轨迹。
*   **标准方程**：
    *   焦点在x轴上：`x²/a² + y²/b² = 1`
    *   焦点在y轴上：`y²/a² + x²/b² = 1` (其中 `a > b > 0`)
*   **基本关系**：`a² = b² + c²`，其中 `c` 是半焦距。
*   **性质**：长轴长为 `2a`，短轴长为 `2b`，顶点，对称性。
*   **离心率 (Eccentricity, e)**：`e = c/a`，取值范围 `0 < e < 1`。`e` 越接近1，椭圆越扁；`e` 越接近0，椭圆越接近圆。

#### **2.3 双曲线**
*   **定义**：平面内与两个定点 `F₁`, `F₂` (焦点) 的距离之差的绝对值为常数 `2a` (且 `0 < 2a < |F₁F₂|`) 的点的轨迹。
*   **标准方程**：
    *   焦点在x轴上：`x²/a² - y²/b² = 1`
    *   焦点在y轴上：`y²/a² - x²/b² = 1`
*   **基本关系**：`c² = a² + b²`。
*   **性质**：实轴长为 `2a`，虚轴长为 `2b`，顶点，对称性。
*   **渐近线 (Asymptotes)**：`y = ±(b/a)x` (焦点在x轴) 或 `y = ±(a/b)x` (焦点在y轴)。
*   **离心率 (e)**：`e = c/a`，取值范围 `e > 1`。`e` 越大，双曲线开口越开阔。

#### **2.4 抛物线**
*   **定义**：平面内与一个定点 `F` (焦点) 和一条定直线 `l` (准线) 的距离相等的点的轨迹。
*   **标准方程** (顶点在原点)：
    *   `y² = 2px` (开口向右)
    *   `y² = -2px` (开口向左)
    *   `x² = 2py` (开口向上)
    *   `x² = -2py` (开口向下)
    (其中 `p > 0`，称为焦参数，焦点到准线的距离为 `p`)

#### **2.5 曲线与方程**
*   **轨迹方程 (Locus Equation)**：介绍求动点轨迹方程的基本步骤（建系、设点、列式、化简、验证）。
*   **参数方程 (Parametric Equation)**：用一个参数（如时间 `t` 或角度 `θ`）来表示曲线上点的坐标 `x` 和 `y`。例如椭圆的参数方程为 `x = a cos(θ)`, `y = b sin(θ)`。
*   **极坐标系 (Polar Coordinate System)**：用极径 `ρ` 和极角 `θ` 来确定点的位置。
*   **坐标转换**：
    *   直角坐标转极坐标：`ρ² = x² + y²`, `tan(θ) = y/x`。
    *   极坐标转直角坐标：`x = ρ cos(θ)`, `y = ρ sin(θ)`。

### **第三章：空间向量及其应用**

本章将平面向量的概念推广到三维空间，并应用于解决立体几何问题。

#### **3.1 & 3.2 空间向量运算与基本定理**
*   **基本概念**：空间向量的定义、模、相等向量、平行向量等与平面向量类似。
*   **共面向量 (Coplanar Vectors)**：可以平移到同一平面内的一组向量。
*   **向量共面定理**：向量 `p` 与两个不共线向量 `a`, `b` 共面的充要条件是存在唯一的实数对 `x, y`，使得 `p = xa + yb`。
*   **空间向量基本定理**：如果三个向量 `e₁`, `e₂`, `e₃` 不共面，那么对空间任一向量 `a`，存在唯一的有序实数组 `x, y, z`，使得 `a = xe₁ + ye₂ + ze₃`。`{e₁, e₂, e₃}` 构成空间的一个基底。

#### **3.3 空间向量的坐标表示**
*   **空间直角坐标系**：引入z轴，构成右手坐标系。
*   **坐标运算**：设 `a = (x₁, y₁, z₁)`，`b = (x₂, y₂, z₂)`
    *   `a ± b = (x₁±x₂, y₁±y₂, z₁±z₂)`
    *   `λa = (λx₁, λy₁, λz₁)`
    *   `a · b = x₁x₂ + y₁y₂ + z₁z₂`
    *   `|a| = √(x₁² + y₁² + z₁²)`
*   **两点间向量与距离**：`P₁(x₁, y₁, z₁)` 和 `P₂(x₂, y₂, z₂)`，则 `P₁P₂→ = (x₂-x₁, y₂-y₁, z₂-z₁)`，距离 `|P₁P₂| = √((x₂-x₁) ² + (y₂-y₁)² + (z₂-z₁)²)`。

#### **3.4 空间向量在立体几何中的应用**
*   **判定位置关系**：
    *   **线线平行/垂直**：方向向量平行/垂直（点积为0）。
    *   **线面平行/垂直**：直线的方向向量 `d` 与平面的法向量 `n` 的关系。`d ⊥ n` ⇒ 线面平行；`d || n` ⇒ 线面垂直。
    *   **面面平行/垂直**：两平面的法向量 `n₁`, `n₂` 的关系。`n₁ || n₂` ⇒ 面面平行；`n₁ ⊥ n₂` ⇒ 面面垂直。
*   **计算角**：
    *   **异面直线夹角**：方向向量夹角的锐角或直角。
    *   **线面角 (θ)**：方向向量 `d` 与法向量 `n` 的夹角为 `α`，则 `sin(θ) = |cos(α)|`。
    *   **二面角 (θ)**：两平面的法向量 `n₁`, `n₂` 夹角的锐角或直角。
*   **计算距离**：
    *   **点到平面距离**：`d = |AP→ · n| / |n|`，其中 `A` 是平面外一点，`P` 是平面内任一点，`n` 是法向量。

### **第四章：数列**

本章研究按一定次序排列的一列数。

#### **4.1 等差数列**
*   **定义**：从第二项起，每一项与它的前一项的差等于同一个常数 `d`（公差）。
*   **通项公式**：`aₙ = a₁ + (n-1)d`。
*   **前n项和公式**：`Sₙ = n(a₁ + aₙ)/2 = na₁ + n(n-1)d/2`。
*   **性质**：若 `m+n = p+q`，则 `aₘ + aₙ = aₚ + a_q`。

#### **4.2 等比数列**
*   **定义**：从第二项起，每一项与它的前一项的比等于同一个常数 `q`（公比），`q ≠ 0`。
*   **通项公式**：`aₙ = a₁ * qⁿ⁻¹`。
*   **前n项和公式**：
    *   当 `q ≠ 1` 时，`Sₙ = a₁(1 - qⁿ) / (1 - q)`。
    *   当 `q = 1` 时，`Sₙ = na₁`。
*   **无穷等比数列求和**：当 `|q| < 1` 时，所有项的和 `S = a₁ / (1 - q)`。

#### **4.3 数列的概念与性质**
*   **基本概念**：有穷数列、无穷数列。
*   **单调性**：增数列、减数列、常数列。
*   **表示方法**：
    *   **通项公式 (Explicit Formula)**：用 `n` 的表达式直接表示 `aₙ`。
    *   **递推公式 (Recurrence Relation)**：给出首项和 `aₙ` 与 `aₙ₋₁` (或前几项) 之间的关系。

#### **4.4 数学归纳法**
*   **原理**：一种证明与正整数 `n` 有关的命题的常用方法。
*   **步骤**：
    1.  **奠基 (Base Case)**：证明当 `n` 取第一个值 `n₀` 时命题成立。
    2.  **归纳递推 (Inductive Step)**：假设当 `n = k` (`k ≥ n₀`) 时命题成立，然后证明当 `n = k+1` 时命题也成立。
*   **应用**：常用于证明数列通项公式、求和公式以及不等式等。

#### **4.5 用迭代序列求√2的近似值**
*   **迭代法 (Iteration Method)**：一种通过重复计算来逼近精确解的算法。
*   **迭代序列**：由递推公式 `xₙ₊₁ = f(xₙ)` 生成的数列。
*   **巴比伦算法 (Babylonian Method)**：求 `√A` 的近似值，其迭代公式为 `xₙ₊₁ = (xₙ + A/xₙ)/2`。书中用此方法展示了如何高效地计算 `√2` 的近似值。
