### **第一章：集合与逻辑 (Sets and Logic)**

本章主要介绍数学的基本语言——集合，以及逻辑推理的基础——常用逻辑用语。

#### **1.1 集合初步 (Introduction to Sets)**

*   **集合 (Set) 与元素 (Element)**
    *   **定义**: 把一些确定的、不同的对象看作一个整体，就形成了集合。集合中的每个对象称为元素。
    *   **特性**:
        1.  **确定性 (Determinacy)**: 给定一个集合，任何一个对象是不是这个集合的元素是确定的。
        2.  **互异性 (Distinctness)**: 集合中的元素是互不相同的。
        3.  **无序性 (Unorderedness)**: 集合中的元素没有固定的顺序。
    *   **表示**:
        *   集合通常用大写字母 A, B, C... 表示。
        *   元素通常用小写字母 a, b, c... 表示。
        *   关系符号: `a ∈ A` (a 属于 A)，`a ∉ A` (a 不属于 A)。

*   **集合的分类与表示**
    *   **分类**: 有限集 (Finite Set) 和无限集 (Infinite Set)。
    *   **空集 (Empty Set)**: 不含任何元素的集合，记作 `∅`。规定空集是任何集合的子集。
    *   **表示方法**:
        1.  **列举法 (Roster Method)**: 将元素一一列出，如 `{1, 2, 3}`。
        2.  **描述法 (Set-Builder Notation)**: 描述集合中元素的共同特征，如 `{x | x 满足性质 P}`。
        3.  **区间表示法 (Interval Notation)**: 用于表示连续的数集，如 `(a, b)` 开区间, `[a, b]` 闭区间。

*   **常用数集**
    *   自然数集 (Natural Numbers): **N**
    *   整数集 (Integers): **Z**
    *   有理数集 (Rational Numbers): **Q**
    *   实数集 (Real Numbers): **R**

*   **集合间的关系与运算**
    *   **子集 (Subset)**: 如果集合 A 中的任何一个元素都是集合 B 的元素，则称 A 是 B 的子集，记作 `A ⊆ B`。
    *   **真子集 (Proper Subset)**: 如果 `A ⊆ B`，且 B 中至少有一个元素不属于 A，则称 A 是 B 的真子集，记作 `A ⊂ B`。
    *   **集合相等 (Set Equality)**: 如果 `A ⊆ B` 且 `B ⊆ A`，则 `A = B`。
    *   **交集 (Intersection)**: 由所有属于集合 A 且属于集合 B 的元素组成的集合，记作 `A ∩ B`。`A ∩ B = {x | x ∈ A 且 x ∈ B}`。
    *   **并集 (Union)**: 由所有属于集合 A 或属于集合 B 的元素组成的集合，记作 `A ∪ B`。`A ∪ B = {x | x ∈ A 或 x ∈ B}`。
    *   **全集 (Universal Set)**: 包含所研究问题中所有元素的集合，通常记作 `U`。
    *   **补集 (Complement)**: 对于全集 U 的一个子集 A，由 U 中所有不属于 A 的元素组成的集合，记作 `∁_U A` 或 `A'`。`∁_U A = {x | x ∈ U 且 x ∉ A}`。

#### **1.2 常用逻辑用语 (Common Logical Terms)**

*   **命题 (Proposition)**
    *   **定义**: 可以判断真假的陈述句。判断为真的称作真命题，判断为假的称作假命题。
    *   **形式**: 常见的命题形式为 “若 p，则 q”，其中 p 是条件，q 是结论。

*   **充分条件与必要条件 (Sufficient and Necessary Conditions)**
    *   **定义**:
        *   若 `p ⇒ q`（由 p 能推出 q），则 p 是 q 的**充分条件**，q 是 p 的**必要条件**。
        *   若 `p ⇔ q`（p 与 q 互为充要条件），则 p 是 q 的**充分必要条件**（简称**充要条件**）。
    *   **集合解释**: 若 `{x|p(x)} ⊆ {x|q(x)}`，则 p 是 q 的充分条件。若两集合相等，则互为充要条件。

*   **反证法 (Proof by Contradiction)**
    *   **描述**: 一种间接证明方法。从否定命题的结论出发，经过一系列正确的逻辑推理，最后引出矛盾，从而证明原结论是正确的。
    *   **经典举例**: 证明 `√2` 是无理数。
        *   **步骤**: 假设 `√2` 是有理数，则 `√2 = m/n` (m, n 为互素的正整数)。平方得 `2 = m²/n²`，即 `m² = 2n²`。所以 `m²` 是偶数，`m` 也是偶数。设 `m = 2k`，代入得 `(2k)² = 2n²`，即 `4k² = 2n²`，`n² = 2k²`。所以 `n²` 也是偶数，`n` 也是偶数。这与 m, n 互素的假设矛盾。故假设不成立，`√2` 是无理数。

### **第二章：等式与不等式 (Equations and Inequalities)**

本章系统研究等式和不等式的性质、解法及其应用，重点介绍基本不等式。

#### **2.1 等式与不等式的性质 (Properties of Equations and Inequalities)**

*   **等式的性质**: 传递性、加法性质、乘法性质。
*   **一元二次方程**:
    *   **解集**: 由判别式 `Δ = b² - 4ac` 的符号决定。
    *   **根与系数的关系 (韦达定理)**: 对于 `ax² + bx + c = 0` 的两根 `x₁, x₂`，有 `x₁ + x₂ = -b/a`，`x₁x₂ = c/a`。

*   **不等式的性质**:
    1.  **传递性**: 若 `a > b` 且 `b > c`，则 `a > c`。
    2.  **加法性质**: 若 `a > b`，则 `a + c > b + c`。
    3.  **乘法性质**:
        *   若 `a > b` 且 `c > 0`，则 `ac > bc`。
        *   若 `a > b` 且 `c < 0`，则 `ac < bc` (不等号方向改变)。
    4.  **同向可加性**: 若 `a > b`, `c > d`，则 `a + c > b + d`。
    5.  **正数同向可乘性**: 若 `a > b > 0`, `c > d > 0`，则 `ac > bd`。

#### **2.2 不等式的求解 (Solving Inequalities)**

*   **一元二次不等式**:
    *   **解法**: 求解 `ax² + bx + c > 0` (或 `<0`)，通常先求对应方程的根，然后结合二次函数图像（开口方向和与x轴的交点）来确定解集。
    *   **口诀**: “大于取两边，小于取中间”（指 `a>0` 且 `Δ>0` 的情况）。

*   **分式不等式**:
    *   **解法**: `f(x)/g(x) > 0` 等价于 `f(x)g(x) > 0`。`f(x)/g(x) ≥ 0` 等价于 `f(x)g(x) ≥ 0` 且 `g(x) ≠ 0`。关键是移项通分，使一侧为0，然后转化为整式不等式求解。

*   **含绝对值的不等式**:
    *   `|x| < a` (a>0)  ⇔  `-a < x < a`
    *   `|x| > a` (a>0)  ⇔  `x > a` 或 `x < -a`
    *   **通用方法**: 分段讨论，根据绝对值内表达式的正负去掉绝对值符号。

#### **2.3 基本不等式及其应用 (Basic Inequalities and Applications)**

*   **基本不等式 (AM-GM 不等式)**:
    *   **定理**: 对于任意两个正数 `a, b`，有 `(a+b)/2 ≥ √ab`。
    *   **等号成立条件**: 当且仅当 `a = b` 时，等号成立。
    *   **几何意义**: 算术平均值不小于几何平均值。
    *   **常用变形**: `a + b ≥ 2√ab` (a, b > 0)。

*   **其他重要不等式**:
    *   `a² + b² ≥ 2ab` (a, b ∈ R)。当且仅当 `a=b` 时取等号。
    *   **三角不等式**: `|a+b| ≤ |a| + |b|`。

*   **应用**: 利用基本不等式求最值。
    *   **“和定积最大”**: 若 `x + y = S` (定值)，则当 `x = y` 时，乘积 `xy` 有最大值 `S²/4`。
    *   **“积定和最小”**: 若 `xy = P` (定值)，则当 `x = y` 时，和 `x + y` 有最小值 `2√P`。
    *   **使用前提**: “一正、二定、三相等”。

### **第三章：幂、指数与对数 (Powers, Exponents, and Logarithms)**

本章将指数的概念从整数推广到实数，并引入其逆运算——对数。

#### **3.1 幂与指数 (Powers and Exponents)**

*   **指数幂的拓展**:
    *   **整数指数幂**: `a⁰ = 1` (a≠0)，`a⁻ⁿ = 1/aⁿ` (a≠0)。
    *   **有理数指数幂**: `a^(m/n) = ⁿ√aᵐ` (a>0)。
    *   **实数指数幂**: 将指数推广到全体实数，运算性质保持不变。
*   **运算性质** (a>0, b>0, r,s∈R):
    1.  `aʳ aˢ = aʳ⁺ˢ` (同底数幂相乘)
    2.  `(aʳ)ˢ = aʳˢ` (幂的乘方)
    3.  `(ab)ʳ = aʳ bʳ` (积的乘方)

#### **3.2 对数 (Logarithms)**

*   **对数的定义**:
    *   如果 `aˣ = N` (a>0, a≠1)，那么数 x 叫做以 a 为底 N 的对数，记作 `x = logₐN`。
    *   `a` 是底数 (base)，`N` 是真数 (argument)。
    *   **重要恒等式**: `a^(logₐN) = N` 和 `logₐaˣ = x`。

*   **常用对数与自然对数**:
    *   **常用对数 (Common Logarithm)**: 以 10 为底的对数，记作 `lg N`。
    *   **自然对数 (Natural Logarithm)**: 以无理数 `e` (≈2.71828) 为底的对数，记作 `ln N`。

*   **对数的运算性质** (a>0, a≠1, M>0, N>0):
    1.  `logₐ(MN) = logₐM + logₐN`
    2.  `logₐ(M/N) = logₐM - logₐN`
    3.  `logₐ(Mⁿ) = n logₐM`

*   **换底公式 (Change-of-Base Formula)**:
    *   `logₐb = (log_c b) / (log_c a)` (a, c > 0 且 ≠ 1, b > 0)。
    *   推论: `logₐb · log_b a = 1`。

### **第四章：幂函数、指数函数与对数函数 (Power, Exponential, and Logarithmic Functions)**

本章系统学习三种基本初等函数。

#### **4.1 幂函数 (Power Function)**

*   **定义**: 形如 `y = xᵃ` (α∈R) 的函数。
*   **图像与性质**:
    *   所有图像都经过点 `(1, 1)`。
    *   当 α > 0 时，在 `(0, +∞)` 上是增函数。
    *   当 α < 0 时，在 `(0, +∞)` 上是减函数。
    *   图像的凹凸性、奇偶性等取决于 α 的具体值。

#### **4.2 指数函数 (Exponential Function)**

*   **定义**: 形如 `y = aˣ` (a>0, a≠1) 的函数。
*   **图像与性质**:
    *   **定义域**: R。
    *   **值域**: `(0, +∞)`。
    *   图像恒过点 `(0, 1)`。
    *   **单调性**:
        *   当 `a > 1` 时，是 R 上的增函数。
        *   当 `0 < a < 1` 时，是 R 上的减函数。
    *   `y = aˣ` 与 `y = (1/a)ˣ` 的图像关于 y 轴对称。

#### **4.3 对数函数 (Logarithmic Function)**

*   **定义**: 形如 `y = logₐx` (a>0, a≠1) 的函数。
*   **图像与性质**:
    *   **定义域**: `(0, +∞)`。
    *   **值域**: R。
    *   图像恒过点 `(1, 0)`。
    *   **单调性**:
        *   当 `a > 1` 时，是 `(0, +∞)` 上的增函数。
        *   当 `0 < a < 1` 时，是 `(0, +∞)` 上的减函数。
    *   `y = logₐx` 与 `y = log_(1/a) x` 的图像关于 x 轴对称。
    *   **关系**: 指数函数 `y = aˣ` 与对数函数 `y = logₐx` 互为**反函数**，它们的图像关于直线 `y = x` 对称。

### **第五章：函数的概念、性质及应用 (Concepts, Properties, and Applications of Functions)**

本章是对函数进行一般性的、抽象的探讨，总结函数的共性。

#### **5.1 函数的概念 (Concept of a Function)**

*   **函数定义**: 设 A, B 是非空的数集，如果按照某种确定的对应关系 f，使对于集合 A 中的任意一个数 x，在集合 B 中都有唯一确定的数 y 和它对应，那么就称 f: A→B 为从集合 A 到集合 B 的一个函数。
*   **三要素**: 定义域 (Domain)、值域 (Range)、对应关系 (Correspondence Rule)。
*   **表示方法**: 解析法、列表法、图像法。
*   **分段函数 (Piecewise Function)**: 在定义域的不同部分，有不同的对应关系。

#### **5.2 函数的基本性质 (Basic Properties of Functions)**

*   **单调性 (Monotonicity)**:
    *   **增函数/减函数**: 描述函数值 `y` 随自变量 `x` 变化的趋势。
    *   **判断方法**: 定义法（取值、作差、变形、定号）、图像法。

*   **奇偶性 (Parity)**:
    *   **前提**: 定义域关于原点对称。
    *   **偶函数 (Even Function)**: `f(-x) = f(x)`，图像关于 y 轴对称。
    *   **奇函数 (Odd Function)**: `f(-x) = -f(x)`，图像关于原点中心对称。

*   **最值 (Extrema)**:
    *   **最大值 (Maximum) / 最小值 (Minimum)**: 函数在其定义域内的最大或最小的函数值。
    *   **求法**: 利用单调性（在单调区间的端点取最值）、基本不等式、二次函数配方法等。

#### **5.3 函数的应用 (Applications of Functions)**

*   **函数零点 (Zero of a Function)**:
    *   **定义**: 使 `f(x) = 0` 的实数 x。
    *   **等价关系**: 方程 `f(x) = 0` 的实数根 ⇔ 函数 `y = f(x)` 的零点 ⇔ 函数 `y = f(x)` 的图像与 x 轴交点的横坐标。
    *   **零点存在性定理**: 如果函数 `y=f(x)` 在区间 `[a, b]` 上的图像是连续不断的，并且 `f(a)f(b) < 0`，则函数在该区间内至少有一个零点。
    *   **二分法 (Bisection Method)**: 一种求函数近似零点的方法，通过不断地把零点所在的区间一分为二，来逼近零点。

#### **5.4 反函数 (Inverse Function) (*选学内容*)**

*   **定义**: 如果函数 `y=f(x)` 中，对于值域中的任意一个 y，在定义域中都有唯一确定的 x 与之对应，则存在反函数，记为 `y = f⁻¹(x)`。
*   **性质**:
    *   原函数的定义域是反函数的值域，原函数的值域是反函数的定义域。
    *   互为反函数的两个函数图像关于直线 `y = x` 对称。
*   **求法**:
    1.  从 `y=f(x)` 中反解出 `x = f⁻¹(y)`。
    2.  将 x 和 y 互换，写成 `y = f⁻¹(x)`。
    3.  注明反函数的定义域。
