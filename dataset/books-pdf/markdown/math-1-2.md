### **第六章：计数原理 (Counting Principles)**

本章系统介绍计数的两个基本原理，并在此基础上深入探讨排列、组合以及二项式定理。

#### **6.1 乘法原理与加法原理**
*   **乘法原理 (Multiplication Principle / 分步计数原理)**：完成一件事需要分 `n` 个步骤，若完成第1步有 `a₁` 种方法，第2步有 `a₂` 种方法，…，第 `n` 步有 `aₙ` 种方法，则完成这件事共有 `a₁ × a₂ × ... × aₙ` 种方法。
*   **加法原理 (Addition Principle / 分类计数原理)**：完成一件事有 `n` 类方法，第1类方法有 `a₁` 种，第2类有 `a₂` 种，…，第 `n` 类有 `aₙ` 种，则完成这件事共有 `a₁ + a₂ + ... + aₙ` 种方法。

#### **6.2 排列 (Permutation)**
*   **定义**：从 `n` 个不同元素中取出 `m` (`m ≤ n`) 个元素，按照一定的顺序排成一列。
*   **排列数 (Permutation Number)**：从 `n` 个不同元素中取出 `m` 个元素的所有排列的个数，记作 P(n, m) 或 A(n, m)。
*   **排列数公式**：
    *   `P(n, m) = n × (n-1) × ... × (n-m+1)`
    *   `P(n, m) = n! / (n-m)!`
*   **全排列 (Full Permutation)**：`n` 个不同元素全部取出的排列，其个数为 `P(n, n) = n!` (n的阶乘)。
*   **规定**：`0! = 1`。

#### **6.3 组合 (Combination)**
*   **定义**：从 `n` 个不同元素中取出 `m` (`m ≤ n`) 个元素，并成一组（不考虑顺序）。
*   **组合数 (Combination Number)**：从 `n` 个不同元素中取出 `m` 个元素的所有组合的个数，记作 C(n, m)。
*   **组合数公式**：`C(n, m) = P(n, m) / P(m, m) = n! / [m!(n-m)!]`
*   **组合数的性质**：
    1.  `C(n, m) = C(n, n-m)`
    2.  `C(n+1, m) = C(n, m) + C(n, m-1)` (杨辉三角/帕斯卡恒等式)

#### **6.5 二项式定理 (Binomial Theorem)**
*   **杨辉三角 (Yang Hui's Triangle)**：揭示了 `(a+b)ⁿ` 展开式系数的规律。
*   **二项式定理**：`(a+b)ⁿ = C(n,0)aⁿ + C(n,1)aⁿ⁻¹b¹ + ... + C(n,r)aⁿ⁻ʳbʳ + ... + C(n,n)bⁿ`
*   **通项公式**：展开式的第 `r+1` 项为 `T(r+1) = C(n, r) aⁿ⁻ʳ bʳ`。
*   **二项式系数的性质**：
    *   所有二项式系数的和为 `2ⁿ`。
    *   奇数项的二项式系数和等于偶数项的二项式系数和，均为 `2ⁿ⁻¹`。
    *   当 `n` 为偶数时，中间一项 `C(n, n/2)` 最大；当 `n` 为奇数时，中间两项 `C(n, (n-1)/2)` 和 `C(n, (n+1)/2)` 相等且最大。

### **第七章：概率初步（续）**

本章在必修内容的基础上，进一步深入探讨概率论中的核心概念，如条件概率、随机变量及其分布。

#### **7.1 条件概率与相关公式**
*   **条件概率 (Conditional Probability)**：在事件A发生的条件下，事件B发生的概率，记为 `P(B|A)`。
*   **公式**：`P(B|A) = P(A∩B) / P(A)`
*   **乘法公式**：`P(A∩B) = P(A) * P(B|A)`
*   **全概率公式 (Total Probability Formula)**：如果事件 `Ω₁, Ω₂, ..., Ωₙ` 构成样本空间的一个划分，则对任意事件A，有 `P(A) = Σ P(A|Ωᵢ)P(Ωᵢ)`。
*   **贝叶斯公式 (Bayes' Theorem)**：用于在已知新信息（事件A发生）后，更新对原因（事件Ωᵢ）的概率判断。`P(Ωᵢ|A) = [P(A|Ωᵢ)P(Ωᵢ)] / P(A)`。它联系了先验概率 `P(Ωᵢ)` 和后验概率 `P(Ωᵢ|A)`。

#### **7.2 随机变量的分布与特征**
*   **随机变量 (Random Variable)**：用数值来表示随机现象结果的变量。
*   **概率分布 (Probability Distribution)**：描述随机变量所有可能取值及其对应概率的表格或函数。
*   **期望 (Expected Value, E[X])**：随机变量取值的加权平均值，权重为对应的概率。`E[X] = Σ xᵢpᵢ`。它反映了随机变量取值的平均水平。
*   **方差 (Variance, D[X])**：度量随机变量取值与其期望值的偏离程度。`D[X] = E[(X - E[X])²] = E[X²] - (E[X])²`。

#### **7.3 常用分布**
*   **二项分布 (Binomial Distribution)** `B(n, p)`：描述 `n` 次独立的伯努利试验中成功次数 `X` 的分布。`P(X=k) = C(n, k) pᵏ (1-p)ⁿ⁻ᵏ`。其期望为 `np`，方差为 `np(1-p)`。
*   **超几何分布 (Hypergeometric Distribution)**：在不放回抽样中，从总数为 `N`（其中 `M` 个为目标）的总体中抽取 `n` 个，抽中 `k` 个目标的概率分布。
*   **正态分布 (Normal Distribution)** `N(μ, σ²)`：一种极其重要的连续型分布，其概率密度函数图像呈钟形。`μ` 是期望（决定对称轴位置），`σ²` 是方差（`σ` 决定曲线的“胖瘦”）。
    *   **3σ原则**：约 `68.3%` 的数据落在 `(μ-σ, μ+σ)` 内，`95.4%` 落在 `(μ-2σ, μ+2σ)` 内，`99.7%` 落在 `(μ-3σ, μ+3σ)` 内。

### **第八章：成对数据的统计分析**

本章关注两个变量之间的关系，介绍相关分析、回归分析和独立性检验。

#### **8.1 成对数据的相关分析**
*   **成对数据**：来自两个相关变量的两组数据。
*   **散点图 (Scatter Plot)**：可视化两个变量之间关系的主要工具。
*   **相关系数 (Correlation Coefficient, r)**：度量两个变量之间**线性**关系强度和方向的指标。
    *   `r` 的取值范围是 `[-1, 1]`。
    *   `|r|` 越接近1，线性关系越强；`|r|` 越接近0，线性关系越弱。
    *   `r > 0` 表示正相关，`r < 0` 表示负相关。

#### **8.2 一元线性回归分析**
*   **回归分析 (Regression Analysis)**：研究变量之间具体数量关系的方法，用于预测和因果分析。
*   **回归直线 (Regression Line)**：用方程 `ŷ = ax + b` 来拟合散点图中的数据趋势。
*   **最小二乘法 (Least Squares Method)**：确定回归系数 `a` 和 `b` 的方法，其原则是使所有观测点到回归直线的竖直距离的平方和（拟合误差）最小。

#### **8.3 2×2列联表**
*   **列联表 (Contingency Table)**：用于展示两个**分类变量**频数分布的表格。
*   **独立性检验 (Test of Independence)**：判断两个分类变量是否相互独立的方法。
*   **卡方检验 (Chi-Squared Test, χ²)**：
    *   **原假设 H₀**：两个变量相互独立。
    *   **计算 χ² 统计量**：`χ² = n(ad-bc)² / [(a+b)(c+d)(a+c)(b+d)]`
    *   **决策**：将计算出的 `χ²` 值与临界值（如显著性水平为0.05时为3.841）比较。若 `χ²` 大于临界值，则拒绝原假设，认为两个变量相关；否则，不拒绝原假设。

### **第九章：数学建模**

本章通过具体的案例，展示将实际问题抽象为数学模型并求解的全过程。

*   **案例1：水葫芦生长率问题**
    *   **过程**：分析数据 → 绘制散点图 → 发现指数增长趋势 → 对数据取对数进行线性化 → 建立线性回归模型 → 将模型还原为指数形式 → 检验模型。
*   **案例2：潜望镜问题**
    *   **过程**：分析物理原理（光的反射定律） → 建立几何模型 → 使用几何或向量方法证明反射定律（反射角等于入射角等）→ 应用模型解决潜望镜中反射镜的安装问题。
*   **建模活动**：提供开放性问题，如“超市收银台的设计”、“出租车临时上下客点的设置”，引导学生进行完整的数学建模实践。
