### **第六章：三角 (Trigonometry)**

本章从任意角的概念出发，系统地定义三角函数，推导常用三角公式，并应用于解三角形。

#### **6.1 正弦、余弦、正切、余切**

*   **任意角与弧度制**
    *   **角的推广**: 将角的概念推广到任意角，包括正角（逆时针旋转）、负角（顺时针旋转）和零角。引入“终边相同角”的概念：与角 α 终边相同的所有角可表示为 `β = α + k·360°` (k∈Z)。
    *   **象限角**: 根据角的终边所在位置，将角分为四个象限角。
    *   **弧度制 (Radian Measure)**: 定义弧长等于半径的圆心角为1弧度。
        *   **换算关系**: `180° = π rad`。
        *   **弧长公式**: `l = |α|r` (α为弧度)。
        *   **扇形面积公式**: `S = (1/2)lr = (1/2)|α|r²`。

*   **任意角的三角函数**
    *   **定义 (单位圆定义)**: 在直角坐标系中，设任意角 α 的顶点在原点，始边与x轴非负半轴重合，终边与单位圆交于点 P(x, y)，则：
        *   正弦 (sine): `sin α = y`
        *   余弦 (cosine): `cos α = x`
        *   正切 (tangent): `tan α = y/x` (x≠0)
        *   余切 (cotangent): `cot α = x/y` (y≠0)
    *   **三角函数线**: 利用单位圆中的有向线段直观表示三角函数值。

*   **同角三角函数的基本关系**
    *   **平方关系**: `sin²α + cos²α = 1`
    *   **商数关系**: `tan α = sin α / cos α`

*   **诱导公式 (Reduction Formulas)**
    *   **目的**: 将任意角的三角函数值转化为锐角的三角函数值。
    *   **口诀**: **“奇变偶不变，符号看象限”**。
        *   `k·(π/2) ± α` (k∈Z)，当 k 为偶数时，函数名不变；当 k 为奇数时，函数名改变（sin↔cos, tan↔cot）。
        *   符号由原函数在 α 视为锐角时所在象限的符号决定。
    *   **常用公式组**:
        *   `sin(2kπ + α) = sin α`, `cos(2kπ + α) = cos α`
        *   `sin(-α) = -sin α`, `cos(-α) = cos α` (奇偶性)
        *   `sin(π - α) = sin α`, `cos(π - α) = -cos α`
        *   `sin(π + α) = -sin α`, `cos(π + α) = -cos α`
        *   `sin(π/2 - α) = cos α`, `cos(π/2 - α) = sin α`
        *   `sin(π/2 + α) = cos α`, `cos(π/2 + α) = -sin α`

#### **6.2 常用三角公式**

*   **两角和与差的三角函数**
    *   `cos(α ± β) = cosαcosβ ∓ sinαsinβ`
    *   `sin(α ± β) = sinαcosβ ± cosαsinβ`
    *   `tan(α ± β) = (tanα ± tanβ) / (1 ∓ tanαtanβ)`

*   **二倍角公式**
    *   `sin(2α) = 2sinαcosα`
    *   `cos(2α) = cos²α - sin²α = 2cos²α - 1 = 1 - 2sin²α`
    *   `tan(2α) = 2tanα / (1 - tan²α)`

*   **辅助角公式**
    *   `asin x + bcos x = √(a²+b²) sin(x+φ)`，其中 `tan φ = b/a`。

*   **半角公式、积化和差、和差化积** (作为变换工具引入)
    *   **半角公式**: `sin²(α/2) = (1-cosα)/2`, `cos²(α/2) = (1+cosα)/2`
    *   **积化和差**: `sinαcosβ = (1/2)[sin(α+β) + sin(α-β)]` 等。
    *   **和差化积**: `sinα + sinβ = 2sin((α+β)/2)cos((α-β)/2)` 等。

#### **6.3 解三角形**

*   **正弦定理 (Law of Sines)**
    *   **内容**: `a/sinA = b/sinB = c/sinC = 2R` (R 为三角形外接圆半径)。
    *   **应用**: 已知两角一边 (AAS/ASA) 或两边和其中一边的对角 (SSA) 解三角形。

*   **余弦定理 (Law of Cosines)**
    *   **内容**:
        *   `a² = b² + c² - 2bccosA`
        *   `cosA = (b² + c² - a²) / (2bc)`
    *   **应用**: 已知三边 (SSS) 或两边夹一角 (SAS) 解三角形。

*   **三角形面积公式**
    *   `S = (1/2)ab sinC = (1/2)ac sinB = (1/2)bc sinA`
    *   **海伦公式**: `S = √[p(p-a)(p-b)(p-c)]`，其中 `p = (a+b+c)/2`。

### **第七章：三角函数 (Trigonometric Functions)**

本章将三角函数作为一类重要的函数进行研究，重点关注其图像和性质。

#### **7.1 & 7.2 正弦函数、余弦函数的图像与性质**

*   **正弦函数 y = sin x**
    *   **图像 (正弦曲线)**: 使用“五点法”作图，关键点为 (0,0), (π/2,1), (π,0), (3π/2,-1), (2π,0)。
    *   **性质**:
        *   **定义域**: R
        *   **值域**: [-1, 1]
        *   **周期性**: 最小正周期 `T = 2π`。
        *   **奇偶性**: 奇函数，图像关于原点对称。
        *   **单调性**: 在 `[-π/2 + 2kπ, π/2 + 2kπ]` 上递增，在 `[π/2 + 2kπ, 3π/2 + 2kπ]` 上递减。

*   **余弦函数 y = cos x**
    *   **图像 (余弦曲线)**: 可由正弦曲线向左平移 π/2 单位得到。关键点为 (0,1), (π/2,0), (π,-1), (3π/2,0), (2π,1)。
    *   **性质**:
        *   **定义域**: R
        *   **值域**: [-1, 1]
        *   **周期性**: 最小正周期 `T = 2π`。
        *   **奇偶性**: 偶函数，图像关于y轴对称。
        *   **单调性**: 在 `[-π + 2kπ, 2kπ]` 上递增，在 `[2kπ, π + 2kπ]` 上递减。

#### **7.3 函数 y = Asin(ωx + φ) 的图像**

*   **参数的物理意义**: 振幅 A、圆频率 ω、初相 φ。
*   **图像变换**:
    1.  **ω**: `y=sinx` → `y=sin(ωx)` (横坐标缩/伸为原来的 1/ω 倍，周期变为 `2π/ω`)。
    2.  **φ**: `y=sin(ωx)` → `y=sin(ωx+φ)` (向左平移 `φ/ω` 个单位)。
    3.  **A**: `y=sin(ωx+φ)` → `y=Asin(ωx+φ)` (纵坐标伸/缩为原来的 A 倍)。

#### **7.4 正切函数的图像与性质**

*   **图像 (正切曲线)**:
    *   由多支不相交的曲线组成。
    *   有垂直于x轴的**渐近线**: `x = π/2 + kπ`。
*   **性质**:
    *   **定义域**: `{x | x ≠ π/2 + kπ, k∈Z}`
    *   **值域**: R
    *   **周期性**: 最小正周期 `T = π`。
    *   **奇偶性**: 奇函数，图像关于原点对称。
    *   **单调性**: 在每个区间 `(-π/2 + kπ, π/2 + kπ)` 上是增函数。

### **第八章：平面向量 (Plane Vectors)**

本章引入向量作为工具，实现了数与形的结合。

#### **8.1 向量的概念和线性运算**

*   **基本概念**:
    *   **向量**: 既有大小又有方向的量。
    *   **模**: 向量的大小或长度。
    *   **零向量**、**单位向量**、**平行向量(共线向量)**、**相等向量**。
*   **线性运算**:
    *   **加法**: 三角形法则、平行四边形法则。满足交换律和结合律。
    *   **减法**: `a - b = a + (-b)`。
    *   **数乘**: `λa`。`|λa| = |λ||a|`。

#### **8.2 向量的数量积 (Scalar Product / Dot Product)**

*   **定义**: `a · b = |a||b|cosθ`，其中 θ 是向量 a 和 b 的夹角。结果是一个**数量**。
*   **几何意义**: 向量 b 在向量 a 方向上的**投影**乘以 a 的模。
*   **性质**:
    *   `a · a = |a|²`
    *   `a · b = b · a` (交换律)
    *   `(λa) · b = λ(a · b)`
    *   `(a + b) · c = a · c + b · c` (分配律)
*   **垂直的判定**: `a ⊥ b` ⇔ `a · b = 0` (a,b为非零向量)。

#### **8.3 向量的坐标表示**

*   **平面向量基本定理**: 如果 `e₁`, `e₂` 是同一平面内的两个不共线向量，那么对于这一平面内的任意向量 `a`，有且只有一对实数 `λ₁`, `λ₂`，使得 `a = λ₁e₁ + λ₂e₂`。
*   **坐标表示**: 在直角坐标系中，`a = (x, y)`。
*   **坐标运算**:
    *   `a ± b = (x₁ ± x₂, y₁ ± y₂)`
    *   `λa = (λx, λy)`
    *   `a · b = x₁x₂ + y₁y₂`
*   **坐标与几何**:
    *   若 `A(x₁, y₁), B(x₂, y₂)`，则 `AB→ = (x₂-x₁, y₂-y₁)`。
    *   `|a| = √(x² + y²)`
    *   `a || b` ⇔ `x₁y₂ - x₂y₁ = 0`。
    *   `a ⊥ b` ⇔ `x₁x₂ + y₁y₂ = 0`。

### **第九章：复数 (Complex Numbers)**

本章将数系从实数拓展到复数，并探讨其运算、几何意义和应用。

#### **9.1 复数及其四则运算**

*   **概念**:
    *   **虚数单位**: `i`，规定 `i² = -1`。
    *   **代数形式**: `z = a + bi` (a, b∈R)。`a`为实部，`b`为虚部。
    *   **复数相等**: `a + bi = c + di` ⇔ `a=c` 且 `b=d`。
    *   **共轭复数**: `z̄ = a - bi`。
*   **四则运算**:
    *   **加减法**: `(a+bi) ± (c+di) = (a±c) + (b±d)i` (实部、虚部分别相加减)。
    *   **乘法**: 像多项式乘法一样运算，再用 `i² = -1` 化简。
    *   **除法**: 将分母实数化，即分子分母同乘以分母的共轭复数。

#### **9.2 复数的几何意义**

*   **复平面**: 用直角坐标系表示复数，x轴为**实轴**，y轴为**虚轴**。复数 `z = a+bi` 对应点 `Z(a, b)` 和向量 `OZ→`。
*   **模**: `|z| = |a+bi| = √(a²+b²)`，表示点 Z 到原点的距离，或向量 `OZ→` 的模。
*   **几何意义**:
    *   **加减法**: 符合向量加减法的平行四边形或三角形法则。
    *   `|z₁ - z₂|`: 表示复平面上对应于 `z₁` 和 `z₂` 的两点之间的距离。

#### **9.3 实系数一元二次方程**

*   在复数范围内，任何实系数一元二次方程 `ax² + bx + c = 0` 都有解。
*   当判别式 `Δ = b² - 4ac < 0` 时，方程有一对**共轭虚根**：`x = (-b ± i√(-Δ)) / (2a)`。

#### **9.4 复数的三角形式 (*选学内容*)**

*   **三角形式**: `z = r(cosθ + isinθ)`
    *   `r = |z|` 是模。
    *   `θ` 是**辐角 (argument)**，`0 ≤ θ < 2π` 的辐角称为**辐角主值**。
*   **乘除运算**: `z₁z₂ = r₁r₂[cos(θ₁+θ₂) + isin(θ₁+θ₂)]` (模相乘，辐角相加)。
*   **棣莫弗定理 (De Moivre's Theorem)**: `[r(cosθ + isinθ)]ⁿ = rⁿ(cos(nθ) + isin(nθ))`。
